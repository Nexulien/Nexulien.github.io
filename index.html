Temporary API+ Documentation

Extra PluginDef features

In a PluginDef, you get the extra parameters nexulien and nexulienBundled. These values are used for the filter by "Nexulien-Exclusives" and "Nexulien-Bundled" search. If the plugin added is uniquely made for Nexulien, please set nexulien to true! If you've got the plugin from somewhere else, and really think it's necessary to contribute this plugin to Nexulien, then set nexulienBundled to true. Otherwise, you can ignore it (ex.: User Scripts).

Extra Slash Command features

If you're on a dev-build, you can make extra debug commands that aren't visible on standalone. You can easily make these "dev-commands" by just setting the devOnly parameter in the command to true!

Extra Margin direction

When using the Margins util object, you get the extra direction all for applying a margin in every direction.
